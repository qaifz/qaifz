## 🚀 Что я умею (Core Competencies)

* **Архитектура автоматизации:** Разработка масштабируемых фреймворков с использованием паттернов проектирования (Screenplay, POM). Уверенная работа с асинхронностью в Python (Event Loop, `pytest-asyncio`).
* **Backend & API Testing:** Тестирование сложных микросервисных архитектур (REST, gRPC). Проверка идемпотентности, работа с брокерами сообщений (Kafka).
* **Databases:** Сложные SQL-запросы в PostgreSQL (Window Functions, CTE, Anti-Joins), понимание индексов (B-Tree), ACID-транзакций и уровней изоляции.
* **CI/CD & Инфраструктура:** Упаковка тестов в Docker (оптимизация слоев, кэширование), настройка пайплайнов (Stages: Checkout, Build, Test, Report, Teardown), безопасная передача секретов.
* **System Design & QA Processes:** Внедрение практик Shift-Left, метрик эффективности автотестов (Defect Escapement, Flaky-rate), планирование нагрузочного и стресс-тестирования.
